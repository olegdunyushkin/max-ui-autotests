# Проект по автоматизации тестирования MAX

UI-автотесты публичных страниц [MAX](https://max.ru), страницы загрузки и веб-формы входа.

> Учебный проект выполнен на Java 17, Selenide и JUnit 5. Тесты не отправляют СМС
> и не используют реальные учётные записи.

## Проверки

- отображение основной информации на главной странице;
- корректность ссылок на магазины приложений;
- корректность служебных и юридических ссылок;
- варианты приложения для Android, iPhone и компьютера;
- отображение формы входа по QR-коду;
- переключение между входом по QR-коду и номеру телефона;
- состояние кнопки входа для номеров разной длины;
- выбор страны и телефонного кода;
- позитивный и негативный поиск страны;
- переключение языка формы входа.

Подробная матрица покрытия находится в [документе по тест-дизайну](docs/test-design.md).

## Локальный запуск

Для запуска требуется Java 17.

```bash
./gradlew clean test -Dheadless=true
```

Запуск только smoke-тестов:

```bash
./gradlew clean test -Dheadless=true -Dtags=smoke
```

Основные параметры:

| Параметр | Значение по умолчанию | Назначение |
|---|---|---|
| `baseUrl` | `https://max.ru` | Адрес главной страницы |
| `webUrl` | `https://web.max.ru` | Адрес веб-версии |
| `browser` | `chrome` | Браузер |
| `browserVersion` | пусто | Версия браузера |
| `browserSize` | `1920x1080` | Разрешение окна |
| `headless` | `false` | Запуск без отображения браузера |
| `remoteUrl` | пусто | Адрес Selenoid |
| `tags` | пусто | Группа запускаемых тестов |

## Отчётность и CI

Проект поддерживает:

- запуск в Jenkins и Selenoid;
- Allure Report;
- скриншоты, Page Source, логи браузера и видео;
- уведомления о результатах в Telegram.

Ссылки и скриншоты будут добавлены после создания и проверки сборки.
